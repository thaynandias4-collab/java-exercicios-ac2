# Exercícios Java - AC2

Lista de exercícios desenvolvida durante a disciplina de Programação em Java da FACENS.

## Tecnologias

- Java
- Eclipse
- VS Code

## Objetivo

Praticar conceitos fundamentais da linguagem Java, incluindo:

- Variáveis
- Estruturas condicionais
- Laços de repetição
- Métodos
- Vetores
- Orientação a Objetos (quando aplicável)
